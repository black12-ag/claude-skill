How it works

Up and running in 3 steps



01 — INSTALL
Run the installer

One curl command clones the repo and runs the full install script. All 17 plugins and 7 marketplaces set up automatically.

curl -fsSL .../bootstrap.sh | bash
02 — LOGIN
Log in to Claude

A browser opens automatically. Sign in with your Anthropic account. The script waits and continues when you are done.

claude login ← auto-triggered
03 — USE
Start using skills

Restart Claude Code. Type / to see every command or /auto to let Claude pick the best skill for your task automatically.

/auto build me a landing page
